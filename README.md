# DDoSolitary Tap

## How do I install these formulae?

`brew install ddosolitary/tap/<formula>`

Or `brew tap ddosolitary/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "ddosolitary/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
